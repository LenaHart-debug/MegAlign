Our Solution

Meet MegAlign

Smart Posture Monitoring

MegAlign is a wearable posture-monitoring device that continuously tracks body position using an ESP32 microcontroller and MPU6050 motion sensor.

When poor posture is detected for more than a few seconds, the device instantly alerts the user through a buzzer and a connected mobile app.

Features

Key technologies that make MegAlign effective.

📡 BLE Connectivity

Connects seamlessly to the mobile app using Bluetooth Low Energy.

🔔 Instant Alerts

Sends immediate buzzer and app notifications when poor posture is detected.

📊 Posture Tracking

Records posture sessions to help users monitor long-term improvement.

⚡ Real-Time Monitoring

Continuously measures body position using the MPU6050 motion sensor.
